# react-native-funnel-chart

Customizable funnel chart component for React Native

## Installation

```sh
npm install react-native-funnel-chart
```

## Usage

```js
import { multiply } from "react-native-funnel-chart";

// ...

const result = await multiply(3, 7);
```
